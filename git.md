# Git

## Install Git

1. Download from [here](https://git-scm.com/download)

2. Select components
> ☒ Additional Icon                \
> ☒ Windows Explorer Integration   \
> ☑ Git LFS                        \
> ☑ Associate .git*                \

3. Select default editor 
> ☑ Visual Studio Code

4. Select line ending convention
> ☑ Checkout as-is, commit Unix-style
